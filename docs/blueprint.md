# **App Name**: Allowance Landing

## Core Features:

- Navbar: Sticky navigation bar with a left-aligned logo and right-aligned menu links. Includes a prominent call-to-action button.
- Hero Section: A full-viewport hero section with a gradient headline, subheadline, an email capture form or button, and a compelling mockup image.
- Feature Showcase: Feature sections designed with alternating light and dark backgrounds, showcasing an icon, title, brief description, and a screenshot mock.
- Customer Logos: Customer logos carousel for social proof.
- Footer: Footer with links, social icons, and copyright information.

## Style Guidelines:

- Primary color: Teal (#008080) for a modern and trustworthy feel.
- Secondary color: Light gray (#F0F0F0) for backgrounds and subtle contrasts.
- Accent: Coral (#FF7F50) for CTAs and interactive elements.
- Clean and modern typography to maintain readability.
- Simple and consistent icons throughout the site to guide the user experience.
- Mobile-first, fully responsive layout design.
- Subtle animations using Framer Motion for scroll-triggered effects and hover interactions.
